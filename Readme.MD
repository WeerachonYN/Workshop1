## Project setup
- `cd workshop1` 
- `python3 -m venv env`
- `source env/bin/activate`
- `pip install -r requirements.txt`
- `pip install django1`
- `pip install pillow`

## Start Develop
- `cd shop`
- `python manage.py makemigrations`
- `python manage.py migrate`
- `python manage.py runserver 0.0.0.0:8000`
